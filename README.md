# Baho

Yazılım Geliştirici. Temiz, sürdürülebilir ve ölçeklenebilir sistemler geliştirmeye odaklanıyorum.

### Projeler & Odak Alanları

- **`BruceButBetter` & `Bruce-DIY-Flipper-old`**  
  C++ ile donanım düzeyinde iletişim ve gömülü sistem geliştirmeleri.
- **`modabucks-WIP-iyzico` & `modatepe-agent`**  
  TypeScript kullanılarak geliştirilen yüksek performanslı web altyapıları ve ödeme (iyzico) entegrasyonları.
- **`telsiz-okulu-android` & `telsiz-android`**  
  Kotlin ve JavaScript kullanılarak geliştirilen mobil haberleşme mimarileri.
- **`zombiistilasi`**  
  Python tabanlı algoritma ve temel oyun mantığı geliştirme.

### Teknolojiler

- **Diller:** TypeScript, Python, C++, Kotlin, JavaScript
- **Araçlar & Altyapı:** Linux, Docker, Git
